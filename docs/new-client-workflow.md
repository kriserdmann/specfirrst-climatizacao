# New Client Workflow

## Por que este arquivo existe

Use este documento quando o projeto for replicado para varios clientes, unidades, marcas ou instancias.

Ele descreve como iniciar uma nova entrega sem reinventar o processo.

## Natureza de Template

Este arquivo e um template. No primeiro chat de escopo, revise se o projeto sera replicado para clientes, unidades ou instancias; ajuste fluxo, dados necessarios e checklist, ou solicite aprovacao humana para remover.

## Fluxo

1. Criar projeto ou instancia a partir da base.
2. Configurar ambiente e secrets.
3. Ajustar identidade, dados e conteudo inicial.
4. Rodar checks de qualidade.
5. Validar com responsavel.
6. Publicar.
