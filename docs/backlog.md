# Backlog

## Por que este arquivo existe

Este documento guarda ideias ainda nao priorizadas. Ele evita misturar desejo futuro com compromisso de implementacao.

Use `docs/issues.md` para trabalho planejado. Use este arquivo para oportunidades, melhorias e hipoteses.

## Natureza de Template

Este arquivo e um template. No primeiro chat de escopo, revise se o projeto precisa de backlog local; ajuste itens, valor esperado, status e observacoes, ou solicite aprovacao humana para remover.

## Itens

| Item | Valor esperado | Status | Observacoes |
| --- | --- | --- | --- |
| `[ideia]` | `[valor]` | Novo | `[nota]` |
